# Testing Directory

This directory contains experimental testing work, performance evaluations, and research experiments.

## Structure

- **`context_windows/`** - Context window performance testing and analysis
- **`pdf_qa_experiment/`** - PDF Q&A evaluation experiments

## Difference from `tests/`

- **`tests/`** - Unit tests, integration tests, and test scripts (pytest, etc.)
- **`testing/`** - Experimental work, performance testing, research, and analysis

## Contents

### Context Windows Testing
- Performance analysis across different context window sizes
- Model comparison (1b vs 3b)
- Visualization and reports

### PDF Q&A Experiments
- Full PDF chat evaluation
- Question-answering accuracy testing
- Model comparison reports
